# GFW-DomainChecker

一个实时检测网站在中国网络环境下可访问性的在线工具，基于Cloudflare Workers构建，帮助用户快速判断网站是否被防火墙屏蔽。

## 功能特点

- **实时状态检测**：精确识别在线、离线、是否被墙一目了然
- **安全管理后台**：密码保护，防止未授权访问
- **响应式设计**：完美适配桌面和移动设备
- **批量监控**：支持同时监控多个网站

## 技术架构

- **运行时**：Cloudflare Workers
- **存储**：Workers KV
- **安全**：SHA-256密码加密
- **前端**：HTML5 + CSS3 + JavaScript

## 快速开始

### 部署要求
- Cloudflare账户
- Workers KV命名空间 - 变量名称是小写的 kv
